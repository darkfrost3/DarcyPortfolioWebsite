Advanced Real-time Rendering Projects

2019-12-13

My projects of Advanced Real-time Rendering in semester Fall 2019.
I implemented 5 projects in this course: Deferred Shading, Moment Shadow Map, Image Based Lighting, Ambient Occlusion and Volume Light
By vulkan

Deferred Shading
8000 local point Lights with deferred shading

Invoke the pixel shader by draw spheres to improve performance

Moment Shadow Map(MSM)
Implement MSM on top of a traditional Shadow Map implementation by blur the shadow map twice in compute shaders.
Without blur

Blur with 3 pixel width

Blur with 5 pixel width

Image Based Lighting(IBL)
Using an HDR environment image as the global light

Teapot with roughness 0.05

Teapot with roughness 0.332

Teapot with roughness 0.8

Ambient Occlusion(AO)
Add an Ambient Occlusion algorithm, with a bilateral filter as an edge-aware blur to smooth out the noise in the calculated ambient occlusion factor.


AO debug view

Volume Light
Implemented volume light by ray marching
