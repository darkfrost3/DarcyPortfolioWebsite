Individua GameEngine Project

2019-09-02

Beat’em up/Game Engine
2018 / Assignment Project

Introduction:
My final project of CS529(Fundamentals of Game Development) in the first semester of Digipen. It’s a component based, data driven game engine using OpenGL to render.
There are two types of enemies and a Boss level in this game. Player has several attack actions and can stop the time when made a just evade.
More levels can be added by just write extra xml files with corresponding art resources.
There is also debug mode, like drawing collision lines, moving camera freely, which can help debug more easily.

Super Spaldy Bros

2019-09-02

Platform game
2017 / Independently by Unity

Introduction:
Super Spaldy Bros is a platform game developed by Unity, containing three mission levels (a Boss level included) and an infinite run mode.

I designed and created a lot of interesting mechanics like a helmet which can knock back falling rocks, dangerous swamps, enemies that will chase player until defeated and so on. And I combined these mechanics into interesting levels intending to bring about a feeling of tense and adventure.

In infinite run mode, the level will go on and on until the player fails. This mode is made of random landforms and enemies, so the experience is different in every single play. In this mode players can repeat to beat their highest score.
