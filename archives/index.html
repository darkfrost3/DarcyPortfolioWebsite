Game Prototype "Pair"

2019-12-14

My game prototype of the engine.
Pair explores the concept of two players seeing different things in a multiplayer game.
Players can choose to play as the Boy or the Girl, and they will be paired with another player randomly over the internet, with no way to communicate except for some simple emotes. The Boy can see monsters but can’t see traps, and the Girl can see traps but can’t see monsters.
They can hold their hands to lead one another through the various hazards as well as to restore HP while fighting enemies.
The game’s intention is to convey the emotion that individuals, when alone, are weak. However, with the help of others, we can overcome any obstacle.

Individua GameEngine Project

2019-09-02

Beat’em up/Game Engine
2018 / Assignment Project

Introduction:
My final project of CS529(Fundamentals of Game Development) in the first semester of Digipen. It’s a component based, data driven game engine using OpenGL to render.
There are two types of enemies and a Boss level in this game. Player has several attack actions and can stop the time when made a just evade.
More levels can be added by just write extra xml files with corresponding art resources.
There is also debug mode, like drawing collision lines, moving camera freely, which can help debug more easily.
